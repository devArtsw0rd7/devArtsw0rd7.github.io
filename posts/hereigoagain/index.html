Here I Go Again

August 23, 2021

Lord, I want to be closer to you.

Here is my attempt to talk with you more regularly. Bless it, and help me in this endeavor.

New post/Git Commands

Generate new post file in ~/blog/prayerblog/:

hugo newposts/NameOfPost.md

Copy path of newly created post in ~/prayerblog/content/posts/

cd content/posts/

code . PATH/TO/POST

Write post in VSCode, Save

In terminal, cd to ~/blog/prayerblog/

Check local changes: hugo server

In browser, navigate to localhost:1313 and confirm additions/changes

Deploy in terminal: ~/blog/prayerblog/

hugo -t contrast-hugo

cd public

In ~/blog/prayerblog/public/

git status

git add .

git commit -m "Commit message here"

git push origin main
